# 部屬本專案

## 使用 Github pages

1. 前往 https://github.com/imkit/imkit-web-sdk
2. 點選右上角 `Fork`，建立副本
3. 參閱 CONFIG.md 修改 `static/config.js` (可選步驟)
4. 修改 `index.html`

```html
<base href="/">
<!-- 修改為 REPO 名稱 -->
<base href="/imkit-web-sdk/">
```

5. 點選 `Settings`
6. 找到 Github pages，選擇 `master branch` 並 `Save`
7. 開啟 `http://{account}.github.io/imkit-web-sdk/` 測試
